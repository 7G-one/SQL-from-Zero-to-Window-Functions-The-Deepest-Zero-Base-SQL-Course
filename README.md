## 课程特色

### 这套课件的深度，比大多数 SQL 教程多走了一步

不只是教你 SELECT。窗口函数、B+ 树索引原理、事务隔离级别、SQL 注入攻防——这些内容通常出现在进阶书籍里，但这里用零基础能懂的方式讲了。

### 生活类比，拒绝黑话

- **数据库** = 带编号的档案柜
- **表** = 柜子里的一个抽屉
- **主键** = 身份证号，唯一标识一行
- **JOIN** = 把两个抽屉的资料对上号
- **索引** = 书的目录，不翻目录也能找内容，但翻了更快
- **事务** = 转账：要么 A 扣钱 B 到账，要么都不动
- **窗口函数** = 在不折叠表格的前提下，给每一行算个"小计"

### 每课都有练习题

看懂了不等于会了。每课末尾都有练习题，做完才算真正学了这课。

### 注释详细到像有人在旁边讲

每条 SQL 语句都有中文注释，不是"这里写了 SELECT"的废话，而是"为什么这么写"、"换个写法会怎样"。

## 如何开始

### 环境准备

本课程使用 SQLite，兼容 MySQL。初学者推荐 SQLite——零配置，装了就能用。

**SQLite：**
```bash
# Windows: 下载 https://www.sqlite.org/download.html
# Mac: brew install sqlite
# Linux: sudo apt-get install sqlite3

# 启动
sqlite3 mydb.db
```

**MySQL：**
```bash
mysql -u root -p
```

### 运行 SQL 文件

**SQLite：**
```bash
sqlite3 mydb.db < 00_零基础入门.sql
# 或在 SQLite 命令行中：
sqlite> .read 00_零基础入门.sql
```

**MySQL：**
```bash
mysql -u root -p mydb < 00_零基础入门.sql
# 或在 MySQL 命令行中：
mysql> source /path/to/00_零基础入门.sql;
```

## 常见问题

**Q: SQLite 还是 MySQL？**
初学者用 SQLite。不用装服务器，数据存在一个文件里，语法和 MySQL 基本兼容。

**Q: 学完能干嘛？**
能设计数据库、写各种查询、优化性能、理解数据库原理、防 SQL 注入。

**Q: 怎么继续深入？**
LeetCode 数据库题练手 → 学一门编程语言（Python/Java）→ 学 ORM 框架 → 学数据库管理（备份、恢复、监控）→ 学分布式数据库。
